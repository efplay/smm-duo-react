# SMM Duo — Portfolio Website

Portfolio website for a social media & content creation duo. Built with React + Vite.

## Tech stack

- React 19
- Vite 8
- CSS (custom, no framework)
- Formspree (contact form)

## Deploy

Deployed on [Vercel](https://vercel.com).
